This extension collects zero personal data. It sends zero data to external servers. All Salesforce API calls go directly from your browser to your own Salesforce org. We have no servers, no analytics, no tracking, and no revenue model based on your data.

What Data Does This Extension Access?

The Salesforce CPQ Toolkit reads Salesforce metadata and object data directly from your own Salesforce org using your existing authenticated browser session. This includes field metadata (for Twin Field Explorer), product records, pricing rules, and CPQ configuration objects (for SKU Quote Explorer).

This data is displayed directly in your browser and never leaves your browser. The extension does not store, log, transmit, or share any of this data with any third party.

External Network Requests

The only network requests made by this extension are:

  • To your own Salesforce org's REST API (*.salesforce.com, *.force.com)
  • To Google Fonts CDN to load the Funnel Sans typeface for display purposes

No other external network requests are made. There is no telemetry, crash reporting, or analytics of any kind.

Authentication & Credentials

This extension uses your browser's existing Salesforce session cookie (the same session your browser tab uses when logged in to Salesforce). No usernames, passwords, API keys, tokens, or OAuth credentials are ever stored, read, or transmitted by this extension.

Data Storage

The only data stored locally by this extension is the URL of your active Salesforce org (stored in chrome.storage.session for the duration of the browser session only). This is cleared automatically when you close your browser.

No personal data, Salesforce records, field metadata, or any other org data is persisted to local storage.

Open Source

The full source code for this extension is publicly available on GitHub. You are encouraged to review it, audit it, and report any security concerns. Transparency is core to this project's values.

Changes to This Policy

Any changes to this privacy policy will be reflected in the extension's GitHub repository and in the extension store listing.
